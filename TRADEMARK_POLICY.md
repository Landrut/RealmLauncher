# Trademark Policy (REALM)

The source code in this repository is open, but the **REALM** brand assets are not licensed for free use.

## Not allowed without written permission
- Using the names `REALM`, `REALM RolePlay`, or confusingly similar names.
- Using REALM logos, icons, banners, artwork, or visual identity.
- Presenting a fork as official, endorsed, or affiliated with REALM.

## Allowed
- Forking and modifying code under the repository license.
- Publishing your fork with your own name, logo, and branding.

## Requirements for forks
- Clearly state that the fork is not an official REALM client.
- Remove/replace all REALM branding from UI, README, releases, and app metadata.
